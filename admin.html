⚙ Ustawienia GitHub
Token PAT przechowywany jest wyłącznie w Twojej przeglądarce (localStorage) – nigdy nie trafia na żaden serwer zewnętrzny.

🔗 Połącz z GitHub

Podłącz swoje repozytorium GitHub, aby zmiany z panelu były automatycznie zapisywane i publikowane na stronie.

1
Wejdź na github.com/settings/tokens/new – zaznacz zakres repo i wygeneruj token.
2
Skopiuj token (zaczyna się od ghp_ lub github_pat_) i wklej go poniżej.
3
Podaj swoją nazwę użytkownika GitHub i nazwę repozytorium, gdzie wgrałeś pliki strony.

💾 Zapisz zmiany do GitHub

Zmiany zostaną opublikowane jako nowy commit. GitHub Pages zaktualizuje stronę w ciągu 1–3 minut.
